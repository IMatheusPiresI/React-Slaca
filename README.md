◽ Clonar o repositório ultilizando Git Bash + o comando:

       $ git clone https://github.com/IMatheusPiresI/React-API-Filmes.git

◽ Acessar a pasta slaca e executar o comando <br>

        $ npm install

◽ Após a instalação das dependências, rodar o comando  <br>

        $ npm start

<h3 id="sobre">📍 Sobre</h3>

✅ Desenvolvido com React-JS, TypeScript, Styled-Components, Context API, React Hooks. <br>
✅ O projeto é uma SPA, trocando apenas o conteúdo dentro da página home.<br>
✅ Dados de criação, remoção e edição dos tópicos do usuário salvados em localStorage para que as continuem salvos após fechar e abrir o navegador <br>
✅ Funcionalidade de tradução do site criada com Context API, relizando a troca do valor e passando para toda a aplicação. <br>
✅ Todas as funcionalidades de criação, remoção e edição dos tópicos criadas com Context API, criando uma unica responsabilidade para cada função, deixando o código mais separado e de fácil manutenção.<br>
✅ Toda estilização criada utilizando Styled-Components<br><br>>

<h3 id="extras">📢 Extras</h3>

✅ Projeto criado com TypeScript para fins didáticos de aprendizado e aplicabilidade de conhecimento para fixação. <br>
✅ Toda a a atualização dos componentes de tópicos são realizadas através da observação do useEffect nas variáveis que recebem o valor para atualizar o componente instantaneamente.<br>
✅ Trabalhei com diversas Context APIS para deixar a responsabilidade das funcionalidades separada para melhor leitura, organização e manutenção de código. <br>
✅ Projeto responsivo. <br>
